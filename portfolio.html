Projeto 1

Compreensor web

O projeto de automação para controle de compressores de ar visa transformar o gerenciamento

desses equipamentos industriais com uma solução inovadora e eficiente. Utilizando o

microcontrolador ESP32, o sistema oferece controle remoto via rede Wi-Fi, o que resulta em uma

redução significativa nos custos operacionais e uma otimização do consumo de energia..
